# Módulo 09 — Processamento de Dados e Fine-Tuning

> De modelos genéricos a especialistas: quando e como customizar LLMs para seu domínio.

## Objetivo

Capacitar o aluno a tomar decisões informadas sobre quando fine-tuning é a abordagem correta, preparar datasets de qualidade, executar treinamentos via API e técnicas eficientes (LoRA/PEFT), e avaliar rigorosamente os resultados.

## Pré-requisitos

- Módulo 02 (APIs e Prompt Engineering)
- Módulo 08 (Arquitetura de Soluções com IA)
- Familiaridade com Node.js/TypeScript

## Conceitos-Chave

- Decision framework: RAG vs Fine-tuning vs Prompt Engineering
- Preparação e validação de datasets JSONL
- Treinamento supervisionado via API
- Parameter-Efficient Fine-Tuning (PEFT/LoRA)
- Métricas de avaliação e testes A/B

## Resultado Esperado

Ao final do módulo, o aluno terá um modelo fine-tunado funcional para um domínio específico, com pipeline reproduzível de dados → treino → avaliação.
