# LinkedIn para Recrutadores

## Como Recrutadores Usam o LinkedIn

- **Boolean search** — combinações de keywords no título e skills
- **Filtros** — localização, experiência, empresa atual/anterior
- **Scan de 6 segundos** — headline → experiência → skills
- **InMail** — mensagem direta para candidatos passivos

## Otimização do Perfil

### Headline (120 chars)

```
❌ "Desenvolvedor na Empresa X"
✅ "AI Engineer | LLMs, RAG, Agents | Python & TypeScript"
```

Inclua: cargo-alvo + tecnologias-chave + diferencial.

### About (2000 chars)

Estrutura:
1. **Hook** — o que você faz e por que importa (2 linhas)
2. **Especialidades** — bullet points com keywords
3. **Resultados** — métricas de impacto
4. **CTA** — como entrar em contato

### Experiência

- Título alinhado com cargo-alvo (não necessariamente o oficial)
- Bullet points com **resultado > responsabilidade**
- Formato: "Verbo + O quê + Resultado mensurável"

### Skills & Endorsements

Top 3 skills devem ser as do cargo-alvo:
- AI/ML Engineering
- LLMs / RAG / Agents
- Python / TypeScript

## Keywords Estratégicas para AI Engineer

```
AI Engineer, Machine Learning, LLM, RAG, Agents,
LangChain, LangGraph, Prompt Engineering, MLOps,
Vector Database, Embeddings, Fine-tuning, NLP,
Python, TypeScript, AWS, Kubernetes
```
