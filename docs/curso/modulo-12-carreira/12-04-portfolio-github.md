# Portfólio GitHub

## O que Recrutadores Avaliam

| Aspecto | O que olham | Tempo gasto |
|---------|-------------|-------------|
| README | Clareza, profissionalismo | 30 segundos |
| Atividade | Commits recentes, consistência | 10 segundos |
| Projetos pinados | Relevância para a vaga | 1 minuto |
| Código | Qualidade, testes, organização | 2-5 minutos (se interessar) |

## Estrutura de Projeto Ideal

```
projeto/
├── README.md          # Hero — vende o projeto
├── docs/              # Documentação detalhada
├── src/               # Código organizado
├── tests/             # Testes (prova de qualidade)
├── .github/           # CI/CD (prova de maturidade)
└── LICENSE            # Profissionalismo
```
