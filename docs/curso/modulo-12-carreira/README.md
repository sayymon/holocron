# Módulo 12 — Carreira e Entrevistas

## Objetivo

Preparar o engenheiro de IA para o mercado de trabalho — desde posicionamento profissional (LinkedIn, portfólio, marca pessoal) até domínio de processos seletivos (entrevistas técnicas, live coding, system design, negociação salarial).

## Pré-requisitos

- Módulos 1-11 concluídos (especialmente o Capstone)
- Projeto funcional para demonstrar em entrevistas

## Resultado Esperado

Ao final deste módulo o aluno será capaz de:
- Posicionar-se como AI Engineer no mercado
- Navegar processos seletivos com confiança
- Negociar compensação de forma informada
- Diferenciar-se em um mercado competitivo
