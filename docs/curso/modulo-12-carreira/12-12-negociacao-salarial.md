# Negociação Salarial

## Princípios

1. **Nunca aceite a primeira oferta** — sempre há margem
2. **Negocie com dados** — pesquisa de mercado ([[12-07-expectativas-salariais]])
3. **Negocie o pacote total** — não só salário base
4. **Timing importa** — negocie depois do "sim", antes de assinar

## Framework de Negociação

```
1. RECEBER A OFERTA
   - Agradecer genuinamente
   - NÃO responder imediatamente
   - "Estou muito animado. Posso analisar e retornar em 2-3 dias?"

2. AVALIAR
   - Comparar com pesquisa de mercado
   - Calcular pacote total (base + bônus + equity + benefícios)
   - Identificar o que é mais importante para você

3. CONTRAPROPOSTA
   - Liderar com entusiasmo pela oportunidade
   - Justificar com dados (mercado, experiência, impacto)
   - Pedir 10-20% acima do que aceitaria

4. FECHAR
   - Aceitar verbalmente + pedir por escrito
   - Confirmar data de início e condições
```

## O que Negociar Além do Salário

| Item | Impacto | Facilidade |
|------|---------|-----------|
| Salário base | Alto | Médio |
| Bônus/variável | Alto | Médio |
| Equity/RSU | Muito alto (longo prazo) | Difícil |
| Remote/híbrido | Alto (qualidade de vida) | Médio |
| Título | Médio (próxima negociação) | Fácil |
| Budget de educação | Médio | Fácil |
| Dias de férias | Médio | Médio |
| Signing bonus | Alto (one-time) | Médio |
| Hardware/setup | Baixo | Fácil |

## Scripts Úteis

### Contraproposta
> "Estou muito empolgado com a oportunidade. Baseado na minha pesquisa de mercado para AI Engineers com minha experiência em [X], e considerando o impacto que posso gerar em [Y], gostaria de propor [valor]. Isso reflete melhor meu posicionamento no mercado."

### Se disserem não
> "Entendo as restrições. Podemos explorar outras formas de compensação? Estou aberto a discutir [bônus/equity/review em 6 meses]."
