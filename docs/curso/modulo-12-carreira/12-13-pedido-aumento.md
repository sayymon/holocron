# Pedido de Aumento

## Quando Pedir

| ✅ Bom momento | ❌ Mau momento |
|---------------|---------------|
| Após entrega significativa | Durante crise na empresa |
| No ciclo de review | Logo após ser contratado |
| Quando responsabilidades cresceram | Sem evidências de impacto |
| Mercado aquecido para sua skill | Quando time está sob pressão |

## Preparação (4-6 semanas antes)

```
1. DOCUMENTAR IMPACTO
   - Projetos entregues com métricas
   - Responsabilidades que cresceram
   - Feedback positivo recebido
   - Problemas que só você resolve

2. PESQUISAR MERCADO
   - Faixa salarial para seu nível atual
   - Ofertas que recebeu (se houver)
   - Pesquisas salariais recentes

3. DEFINIR ASK
   - Valor específico (não "um aumento")
   - Justificativa clara
   - Alternativas aceitáveis (título, benefícios, equity)
```

## Framework da Conversa

```
1. CONTEXTO
   "Nos últimos X meses, assumi [responsabilidades] e entreguei [resultados]"

2. EVIDÊNCIA
   "Especificamente: [projeto A → resultado], [projeto B → resultado]"

3. MERCADO
   "Pesquisando o mercado, profissionais com meu perfil estão em [faixa]"

4. ASK
   "Gostaria de discutir um ajuste para [valor/faixa]"

5. ABERTURA
   "Estou aberto a discutir o melhor caminho para chegarmos lá"
```

## Se a Resposta for "Não Agora"

- Pergunte: "O que preciso demonstrar para chegarmos lá?"
- Peça timeline: "Podemos revisitar em X meses?"
- Documente o acordo por escrito (email de follow-up)
- Avalie se vale esperar ou buscar externamente
