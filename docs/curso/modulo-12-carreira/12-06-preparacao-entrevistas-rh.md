# Preparação Entrevistas RH

## O que RH Avalia

| Dimensão | Perguntas típicas |
|----------|-------------------|
| Motivação | "Por que essa empresa?" "Por que IA?" |
| Cultura | "Como lida com conflito?" "Trabalho em equipe?" |
| Comunicação | Clareza, concisão, estrutura |
| Red flags | Falar mal de empregadores, arrogância |

## Método STAR

```
S — Situation (contexto breve)
T — Task (seu papel específico)
A — Action (o que VOCÊ fez)
R — Result (resultado mensurável)
```

### Exemplo

> "Conte sobre um desafio técnico que superou"

- **S**: Sistema de RAG com latência de 8s em produção
- **T**: Responsável por reduzir para <2s sem perder qualidade
- **A**: Implementei cache de embeddings + hybrid search + reranking
- **R**: Latência caiu para 1.2s, satisfação do usuário subiu 40%

## Perguntas Frequentes + Framework

| Pergunta | Framework |
|----------|-----------|
| "Fale sobre você" | Presente → Passado → Futuro (2 min) |
| "Por que sair do emprego atual?" | Positivo (busco X) > Negativo (fujo de Y) |
| "Maior fraqueza?" | Real + ação de melhoria |
| "Onde se vê em 5 anos?" | Alinhado com a vaga + ambição realista |
| "Pretensão salarial?" | Faixa pesquisada + "negociável" |

## Preparação

```
□ Pesquisar empresa (produto, cultura, notícias recentes)
□ Preparar 5 histórias STAR (liderança, conflito, falha, sucesso, aprendizado)
□ Preparar 3 perguntas para o entrevistador
□ Testar setup técnico (câmera, mic, internet)
□ Vestir-se um nível acima do dress code da empresa
```
