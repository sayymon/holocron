# Networking Estratégico

## Princípio Fundamental

> Networking não é pedir emprego. É **construir relacionamentos** antes de precisar deles.

## Estratégias por Canal

### Online

| Canal | Ação | Frequência |
|-------|------|-----------|
| LinkedIn | Comentar, postar, conectar | Diário |
| Twitter/X | Threads técnicas, engajar com builders | 3x/semana |
| Discord/Slack | Comunidades de IA (LangChain, AI Engineer) | Semanal |
| GitHub | Contribuir em projetos open-source | Mensal |

### Presencial

| Evento | Valor | Dica |
|--------|-------|------|
| Meetups locais | Conexões próximas | Vá com pergunta preparada |
| Conferências | Visibilidade + aprendizado | Fale com speakers depois |
| Hackathons | Demonstrar skills ao vivo | Forme time diverso |

## Framework de Networking

```
1. Dar antes de pedir (compartilhe conhecimento)
2. Ser específico (não "vamos tomar um café")
3. Follow-up em 48h (mensagem personalizada)
4. Manter contato (não só quando precisa)
5. Conectar pessoas (seja um hub)
```

## Cold Outreach que Funciona

```
Estrutura:
- Contexto (por que essa pessoa)
- Valor (o que você oferece/admira)
- Ask específico (pergunta concreta, não "me ajude")
- Facilitar (horário flexível, call curta)
```

## Métricas de Networking

- Conexões relevantes adicionadas/mês
- Conversas 1:1 realizadas/mês
- Conteúdo compartilhado/semana
- Referrals recebidos (indicador de qualidade)
