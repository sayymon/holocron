# Módulo 07 — Ferramentas de IA para Gestão de Projetos

> Como usar IA para transformar planejamento, execução e governança de projetos de software.

## Objetivo

Capacitar o aluno a aplicar ferramentas e técnicas de IA em todo o ciclo de gestão de projetos — do planejamento ao report executivo — automatizando tarefas repetitivas, melhorando estimativas e acelerando decisões.

## Pré-requisitos

- Módulo 01 (Fundamentos de IA)
- Módulo 02 (APIs e Prompt Engineering)
- Experiência básica com ferramentas de gestão (Jira, Notion, etc.)

## Projeto Integrador

Construir um **PM Copilot** que:
1. Recebe requisitos em linguagem natural e gera épicos/histórias
2. Prioriza backlog automaticamente (RICE/WSJF)
3. Gera status reports adaptados por audiência
4. Integra com pelo menos uma ferramenta (Jira, Notion ou Slack)
