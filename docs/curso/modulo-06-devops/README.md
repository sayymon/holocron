# Módulo 06 — Ferramentas de IA para DevOps

> De infraestrutura manual para operações autônomas assistidas por IA.

## Visão Geral

Este módulo explora como LLMs, agentes e técnicas de IA generativa transformam práticas de DevOps — desde geração de IaC até auto-remediação segura, passando por observabilidade inteligente e FinOps automatizado.

- Módulo 01 (Fundamentos de IA)

Ao final do módulo, o aluno será capaz de construir um agente DevOps completo que gera IaC, gerencia clusters Kubernetes, diagnostica problemas, otimiza custos e executa remediações com supervisão humana.
