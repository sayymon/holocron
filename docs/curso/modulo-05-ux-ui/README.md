# Módulo 05 — Ferramentas de IA para UX & UI

> Como a IA está transformando o design de interfaces: da ideação à implementação, passando por prototipação, automação de testes e integração de lógica inteligente.

## Visão Geral

Este módulo explora o ecossistema de ferramentas de IA aplicadas ao ciclo completo de UX/UI — desde a concepção visual até a entrega de código funcional com inteligência embarcada.

| # | Título | Dificuldade | Foco |

| 2 | [[unidade-02-prototipacao-geracao-ui]] | Intermediário | Text-to-UI, Firebase Studio, Figma→código |
| 3 | [[unidade-03-agentes-cli-frontend]] | Intermediário | Gemini CLI, scaffolding, fluxo de trabalho |

## Pré-requisitos

- Conhecimento básico de [[frontend]] (HTML, CSS, JavaScript)
- Familiaridade com [[react]] ou frameworks similares
- Noções de [[prompt-engineering]]
- Conta Google/Firebase (para labs práticos)

## Conceitos-Chave

- [[design-system]] — sistemas de design como base para geração de UI
- [[text-to-ui]] — geração de interfaces a partir de linguagem natural
- [[mcp]] — Model Context Protocol para integração de ferramentas
- [[agentes-de-ia]] — automação inteligente de fluxos de trabalho
- [[rag]] — busca semântica para interfaces inteligentes
