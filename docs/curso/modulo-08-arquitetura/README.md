# Módulo 08 — Arquitetura de Sistemas com IA

> Como projetar sistemas que têm IA como cidadão de primeira classe, não como feature bolted-on.

## Visão Geral

Este módulo cobre desde fundamentos de arquitetura AI-First até padrões enterprise de produção. O foco é capacitar o engenheiro a tomar decisões arquiteturais informadas quando IA é componente central do sistema.
