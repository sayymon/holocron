# Módulo 04 — Criação de Agentes Autônomos

> De chatbots reativos a agentes que planejam, executam, refletem e colaboram.

## Visão Geral

Este módulo cobre o ciclo completo de construção de agentes autônomos: desde a arquitetura fundamental (loop, memória, ferramentas) até sistemas multi-agente com orquestração sofisticada. O foco é prático — cada conceito é implementável com LangGraph e o ecossistema TypeScript.

## Pré-requisitos

- Módulo 02 — APIs e Prompt Engineering (function calling básico)
- Módulo 03 — MCP (Model Context Protocol)
- Familiaridade com TypeScript e async/await

## Resultado Esperado

Ao final deste módulo, o aluno será capaz de projetar, implementar e operar agentes autônomos com memória persistente, múltiplas ferramentas, observabilidade e colaboração multi-agente.
