# Módulo 02 — APIs de IA Generativa e Prompt Engineering

> De consumidor de IA a arquiteto de soluções inteligentes.

## Objetivo

Dominar o ecossistema de APIs de IA generativa, desde a escolha do provedor até a integração robusta em produção, passando por engenharia de prompts avançada, RAG e modelos multimodais.

## Pré-requisitos

- Módulo 01 (Fundamentos de IA) concluído
- Conta em pelo menos um provedor (OpenAI ou Anthropic)
- Node.js 20+ instalado

## Projeto Integrador

Construir uma API back-end que integra OpenAI com RAG, cache inteligente e observabilidade via LangFuse.
